# Scripts

| Script | Description | Location |
|--------|-------------|----------|
| `run_conversion` | Convert a PDF to DOCX | `scripts/run_conversion/run_conversion.ts` |
| `run_compressions` | Compress a PDF at light/balanced/extreme levels | `scripts/run_compressions/run_compressions.ts` |
| `compare_pdf_docx` | Compare a PDF against a DOCX for text, style, and image fidelity | `scripts/compare_pdf_docx/compare_pdf_docx.py` |
| `analyze_columns` | Compare column geometry between a PDF and one or more DOCX files | `scripts/analyze_columns/analyze_columns.py` |
| `debug_column_widths` | Show right-edge distribution of body lines in a PDF column | `scripts/debug_column_widths/debug_column_widths.py` |
| `pdf_compare` | Deep per-page, per-image, per-font analysis across PDF files | `scripts/pdf_compare/pdf_compare.py` |
